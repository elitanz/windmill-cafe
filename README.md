# Windmill Cafe — website

A custom one-page website for **Windmill Cafe**, an old-fashioned family diner on
Highway 13 in Savage, Minnesota (homestyle breakfast & lunch, open daily 6am–2pm).

Built from real public data (Google/Yelp/Tripadvisor listings, Heavy Table review).
Features a scroll-scrub video hero (real griddle footage played frame-by-frame on
scroll), a real menu, photo gallery, verbatim reviews, hours, and an embedded map.

- **Live:** https://elitanz.github.io/windmill-cafe/
- Static HTML/CSS/JS — no build step. Just open `index.html`.
- Vintage-diner identity: cream + diner-red palette, checkerboard motif, Pacifico
  script wordmark.

*Independent sample site. Not affiliated with the business.*
